HTML Grundlagen

Überblick

HTML ist eine Auszeichnungssprache.
Sie wird verwendet um den Inhalt einer Website zu erstellen und zu strukturieren. Mit HTML kann man z.B. Überschriften, Texte, Bilder oder Links einfügen.

Wichtige HTML Elemente

Überschriften

Mit den h-Elementen kann man Überschriften erstellen. Es gibt die Größen h1 bis h6. h1 ist die größte Überschrift und h6 die kleinste.

Div Elemente

Das div-Element wird verwendet um Bereiche oder Container zu erstellen. Es wird oft genutzt um Inhalte zu gruppieren.

Das div selbst sieht man nicht direkt. Es hilft dabei Inhalte zu strukturieren.
