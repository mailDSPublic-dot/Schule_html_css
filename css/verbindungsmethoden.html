Überblick

CSS ist eine Beschreibungssprache. Sie wird verwendet um das Aussehen der Html Bausteine verändern zu können.
HTML erschafft z.B. einen Paragraphen, und CSS ist dann dafür zuständig, dass die Schriftfarbe schwarz ist.

Einbindungsmethoden

Man kann CSS über drei verschiedene Methoden einbinden. Das heiß man kann den CSS Code an drei Orte schreiben.

Inline CSS

Der erste Ort wo man CSS schreiben kann ist direkt beim Element, dass man verändern möchte. Der Vorteil ist, dass es super schnell geht und man nicht groß nachdenken muss.
Der Nachteil ist allerdings, dass man den Code nicht wiederverwenden kann, man Struktur mit Design vermischt und es auf Dauer unübersichtlich wird.

Internal CSS

Der 2. Ort ist im Head der HTML Datei. Da hat man im Gegenzug zum Inline CSS den Vorteil, dass man Styles wiederverwenden kann und man Struktur und Design teilweise getrennt hat
Allerdings ist HTML und CSS dann trotzdem wieder gemischt. Zudem kann man das Layout von CSS dann nicht für mehrere Seiten verwenden.
Diese Methode ist gut für kleine Seiten und und erste Projekte, allerdings ist es nachteilhaft wenn man größere Websites baut.

Hier sieht man den head des HTML Codes. Dort kann man dann mit style festlegen, dass der folgende Code CSS Code ist.
Dort haben wir dann festgelegt, dass alle p Elemente die Farbe Rot haben und die Größe 20 px beträgt. Es macht genau das gleiche wie der Inline CSS Code, aber jetzt für alle p Elemente.

External CSS

Die 3. und die deutlich am meisten verwendete Möglichkeit ist die External CSS Einbindungsmethode. Dort schreibt man eine eigene Datei, wo nur CSS Code steht.
Der Vorteil ist, dass man alle sauber trennt und man auch immer zwischen HTML und CSS hin und her wechseln kann, ohne jedes mal an den Anfang der Datei scrollenzu müssen.

Man schreibt in den Head Abschnitt des HTML Codes nur die eine Zeile und beide Dateien sind miteinander verbunden. rel = "stylesheet" sagt, dass das der CSS Code ist.
href = "style.css" bedeutet, dass die Datei, die style.css heißt den Code beinhaltet. Die Vorraussetzung ist natürlich, dass man eine Datei die style.css heißt vorher erstellt hat.
In der CSS Datei (style.css) kann man dann einfach wieder den CSS Code schreiben.

Dies macht genau das gleiche wie das Internal CSS. Es färbt alle p Elemente Rot und setzt die größe auf 20px.
