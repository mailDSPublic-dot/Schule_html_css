CSS - Selektoren

Was ist ein Selektor?

CSS‑Selektoren bestimmen, welche HTML‑Elemente von einer bestimmten Regel angesprochen werden. Sie ermöglichen es, gezielt einzelne Elemente, Gruppen von Elementen oder bestimmte Zustände wie Hover oder Fokus zu stylen.

Element-Selektoren

Mit den Element-Selektoren können alle Elemente eines bestimmten Typs angewählt werden. Das Folgende Beispiel ändert die Farbe von allen p Elementen auf Rot und setzt die Größe auf 20px.

Klassen-Selektoren

Mit den Klassen-Selektoren kann man alle Elemente einer bestimmten Klasse ansprechen. Dazu muss man vorher im HTML Dokument dann Klassen vergeben. In CSS spricht man diese dann mit einem Punkt vor dem Klassennamen an.
