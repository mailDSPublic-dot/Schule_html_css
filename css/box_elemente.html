CSS - Box Elemente

Was sind Box Elemente?

Boxelemente bilden das grundlegende Layout‑Modell im Web, da jedes HTML‑Element als rechteckige Box dargestellt wird. Jede Box besteht aus Inhalt(Content), Abstand zum Rand (Padding), Rand(Border) und Abstand(Margin) zu anderen Elemenren, diese zusammen bestimmen, wie viel Platz ein Element einnimmt. Das Box‑Modell ist entscheidend, um Abstände, Größen und Layouts präzise zu kontrollieren. Ohne ein Verständnis der Boxelemente wäre modernes, strukturiertes Webdesign kaum möglich. In dem Artikel beschäftigen wir uns erstmal nicht mit dem Content sondern mit den anderen 3.

Das Padding sagt, wie viel Abstand vom Content bis zur Border ist. Man kann dabei jede der 4 Seiten einzeln oder einen Abstand, welcher auf allen 4 Seiten gleich gilt festlegen.

Border

Mit dem border Element kann man Grundlegend verändern, wie der Rand aussieht.
Es besteht aus 3 Parametern: Dicke, Design, Farbe.

Dicke

Mit dem ersten Parameter des border Elements bestimmt man, wie dick die Linie ist, welche um das Element geht Diese kann man mit vielen verschiedenen Einheiten angeben. Einfacherheitshalber nehmen wir jetzt px was Pixel darstellt.
CSS: 

p {
    border: 10px, solid, red;
}

Ich bin 10px dick umrandet!

Ich bin 1px dick umrandet!

Style

Der zweite Parameter ist dafür zuständig, welche Form die Linie hat. Dafür hat man eine große Auswahl. Die möglichen Typen sind: none, hidden, dotted, dashed, solid, double, groove, ridge, inset und outset Für mehr kann man sich auf der Offiziellen Documentation weiter informieren. Bis jetzt sind alle typen solid.
CSS:

p {
    border: 3px, dashed, red;
}

Ich bin umstrichelt!

p {
    border: 10px, groove, red;
}

Ich bin 3D!

Farbe

Der dritte Parameter ist die Farbe. Dort hat man zum einen die meisten Farben (natürlich auf Englisch benannt) und unter anderem auch rgb(red, grenn, blue). Dort kann man aus den drei Farben jede belibige mischen. Dabei kann jede Farbe bis zu 255 gehen.

p {
    border: 3px, double, green;
}

Daraus entstehe ich!

p {
    border: 3px, dotted, rgb(0, 255, 0);
}

Das gleiche in Grün

Margin

Margin ist dafür da, um den Abstand zu den anderen Elementen zu regeln. Damit kann man den Abstand vom Element zu den verscheiden Seiten angeben.

Wie beim Padding hat man auch hier zwischen der gesamten Seiten und den einzelnen wählen.
CSS:

p {
    marign: 100px;
}

Ich habe auf allen Seiten mindestens 100px Abstand zu allem, es kann natürlich auch mehr sein, aber es sind immer mindestens 100 Pixel.
